このプログラムは整数計画問題における0-1ナップザック問題を分枝限定法によって解くプログラムである．

## 概要
数理最適化のアルゴリズムを理解するために、線形計画法の基礎となるシンプレックス法と、整数制約を扱うための分枝限定法を組み合わせて実装．

- **手法**分枝限定法 (Branch and Bound)
- **基礎アルゴリズム**: シンプレックス法 (Simplex Method)
- **言語**: C言語

## 特徴
1. **動的な定数定義**:
   - `NUM_VARS`: 最大化問題における変数の数を定義．
   - `NUM_SLACKS`: シンプレックス法におけるスラック変数の数を定義．
2. **効率的な探索**:
   - 分枝操作によって探索空間を分割し、限定操作（下限・上限の比較）によって不要な探索を削減（枝刈り）するロジックを組み込んでいる．

## 使い方
解きたいナップザック問題をbnb.cのinitial_data[ROWS][COLS]に代入して配列を作成する．このとき定数の定義をすること．
コンパイルはvscodeのコードランナーを用いて
Makefileに以下を入力すると容易である．

CC = gcc
CFLAGS = -Wall -g
LIBS = -lm

# ターゲット名
TARGET = main

# ソースファイル（ここを変えると実行するファイルが変わる）
SRCS = bnb.c simplex.c

# 実行ファイルの生成（ビルド）
$(TARGET): $(SRCS)
	$(CC) $(CFLAGS) $(SRCS) -o $(TARGET) $(LIBS)
  
# 実行用ターゲット
run: $(TARGET)
	./$(TARGET)
# --- ここまで追加 ---

# クリーンアップ
clean:
	rm -f $(TARGET)

## 展望
目的関数と制約式を入れたら自動でスラック変数とその制約条件とかの式を作ってくれるものがあったら便利かも．
